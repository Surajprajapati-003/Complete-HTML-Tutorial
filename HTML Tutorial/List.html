HTML Lists

Lists are used to organize information in a structured way. HTML supports three types of lists:

1. Ordered List (<ol>)

Items are numbered automatically.

  1. Introduction to HTML
  2. HTML Headings
  3. HTML Paragraphs
  4. HTML Links and Media
  5. HTML Lists

2. Unordered List (<ul>)

Items are marked with bullets.

  • Apples
  • Bananas
  • Oranges
  • Mangoes

3. Description List (<dl>)

Used to describe terms and their definitions.

Attributes of Lists

  • type: Changes the marker style of lists (numbers, letters, roman numerals).
  • start: Starts an ordered list from a specific number.
  • reversed: Reverses the order of an ordered list.
