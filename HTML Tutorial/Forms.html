HTML Forms

Forms are used to collect user input on a webpage.

Sample Registration Form

Notes:

  • action – URL where form data is sent.
  • method – HTTP method: get or post.
  • Use required for mandatory fields.
  • Different input types: text, email, password, radio, checkbox.
  • Use <select> and <textarea> for dropdowns and multi-line input.
  • Always include labels with for attribute for accessibility.
